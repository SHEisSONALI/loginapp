# Full Stack Authentication System

A full-stack authentication application built with React, Node.js, Express, PostgreSQL, JWT, and bcrypt.

## Features

- User Registration (Signup)
- User Login
- JWT Authentication
- Protected Routes
- User Dashboard
- Logout Functionality
- Password Reset Flow
- Email Verification Flow
- Refresh Tokens
- Role-Based Access Control (RBAC)
- Last Login Tracking

## Tech Stack

### Frontend
- React
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

### Security
- JWT
- bcrypt

## Project Structure

```text
loginapp
├── frontend
├── backend
└── database
```

## Running the Project

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```
